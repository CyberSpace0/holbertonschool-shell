# _printf

## 📌 Description

This project is a custom implementation of the standard C `printf` function.

It handles basic format specifiers using variadic functions and function pointers.

---

## ⚙️ Features

- %s → String
- %c → Character
- %d → Signed decimal
- %i → Integer

---

## 📥 Prototype

```c
int _printf(const char *format, ...);
```
